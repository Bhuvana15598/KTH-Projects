Thin-Film Temperature Sensor Fabrication & Characterization

This project focuses on the fabrication, post-processing, and characterization of a MEMS-based thin-film temperature sensor using microfabrication techniques. The sensor was fabricated on a silicon wafer through photolithography, oxidation, metal deposition, lift-off, and DRIE processes, followed by experimental characterization using a Wheatstone bridge setup.

Key Highlights:

1. Fabricated a thin-film aluminum RTD temperature sensor.
2. Performed photolithography, oxidation, sputtering, lift-off, and DRIE.
3. Characterized sensor performance using Wheatstone bridge measurements.
4. Evaluated sensitivity, TCR, hysteresis, accuracy, and response time.
5. Achieved experimental TCR values close to the theoretical value for aluminum.

Technologies:

1. MEMS Fabrication
2. Photolithography
3. Thin-Film Deposition
4. Deep Reactive Ion Etching (DRIE)
5. Wheatstone Bridge Characterization
6. LabVIEW Data Acquisition

Outcome:

The study demonstrated the complete MEMS sensor development workflow, from cleanroom fabrication to performance evaluation, highlighting the relationship between fabrication processes and sensor characteristics.
