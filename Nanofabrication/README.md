CMOS Contact Hole Formation and Nanofabrication Process

This folder contains a nanofabrication laboratory study on CMOS contact hole formation using a pre-processed silicon wafer. The project covers key semiconductor manufacturing processes including deposition, photolithography, dry etching, and wafer characterization.

Overview:

1. CMOS wafer processing and contact hole fabrication
2. Thin-film deposition using ALD and PECVD
3. Photolithography and pattern transfer
4. Reactive Ion Etching (RIE) for feature formation
5. Wafer inspection and defect analysis
6. Thickness characterization using spectroscopic ellipsometry
   
Key Technologies:

1. Silicon-on-Insulator (SOI) CMOS
2. Atomic Layer Deposition (ALD)
3. Plasma Enhanced Chemical Vapor Deposition (PECVD)
4. Photolithography
5. Reactive Ion Etching (RIE)
6. Spectroscopic Ellipsometry
